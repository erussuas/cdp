# CDP 2026 Readiness Assessment App v0.1.9

Streamlit MVP for CDP readiness assessment, action planning, Gantt-style timeline, Climate Score Simulator, and Excel import/export.

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Streamlit Cloud
Upload `app.py`, `requirements.txt`, and this README. Do not upload `__pycache__` or local virtual environment folders.
